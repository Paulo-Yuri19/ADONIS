# ADONIS - Premium Fitness Management

## Descrição do Projeto
O Adonis é um sistema de gerenciamento fitness de alta performance que conecta Personal Trainers e Alunos. A plataforma permite validação de profissionais via CREF, envio de solicitações de vínculo, prescrição de treinos e dietas em tempo real, além de chat interno e acompanhamento antropométrico.

## Tecnologias Utilizadas
* **Frontend:** HTML5, CSS3, JavaScript Vanilla, Chart.js.
* **Backend:** Node.js com framework Express.
* **Banco de Dados:** MySQL (Relacional).
* **Segurança:** Autenticação via JWT (JSON Web Token) e criptografia de senhas com Bcrypt.
